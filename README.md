## Présentation

GeoNat est une application qui utilise Vue.js pour le frontend, Bootstrap pour le style et Open Layers pour la cartographie.
L'objectif de l'application est d'explorer, analyser et protéger les espaces naturels sensibles.

## Technologies utilisées

- [Vue.js](https://vuejs.org/)
- [Bootstrap](https://getbootsrap.com/)
- [OpenLayers](https://openlayers.org/)

## Installation

Assurez-vous d'avoir [Node.js](https://nodejs.org/) installé avant de commencer.

```bash
# Cloner le dépôt
git clone https://github.com/SylvieHoareau/GeoNat.git

# Accéder au répertoire frontend 
cd GeoNat/frontend

# Installer les dépendances
npm install

# Démarre l'application Vue.js
npm run serve

```
## Contribuer

Si vous souhaitez contribuer à GéoNat, suivez ces étapes : 
1. Fork le proket
2. Créer une branche pour votre fonctionnalité :


3. Committez vos modifications


5. Ouvrez une demande de fusion (Pull Request)
